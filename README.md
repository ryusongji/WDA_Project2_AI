# 인공지능과 머신러닝 딥러닝

인공지능(Artificial Intelligence)
사고나 학습 등 인간이 가진 지적 능력을 컴퓨터를 통해 구현하는 기술 입니다.

기계학습(Machine Learning)
기계 학습이란 스스로 성능을 향상 시키는 컴퓨터 알고리즘을 만드는 연구 입니다.

딥러닝(Deep Learning)
